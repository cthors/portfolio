Website Design

These are a couple pages from another personal website I designed and created. I coded the site using vanilla JavaScript, html, and CSS.

The repeating pixel graphics were generated programmatically, then saved as image files and added to the site. I created them with the Gnu Image Manipulation Program, using a script I wrote in GIMP's scripting language, Script-Fu (a dialect of Scheme).

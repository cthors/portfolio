Sculptural Lamp

This is a lamp I created by repurposing floating block shelves, napkin holders, and plastic duotangs. Addressable WS2812B LED strips are attached to the inside and diffused by the translucent plastic. The LEDs are run by an ESP8266 microcontroller using WLED, an open-source library that allows for WIFI control of the lights. I repurposed an old power supply from a TV cable box, and soldered a basic circuit including a fuse, a switch, and some capacitors for smoothing out the power.
